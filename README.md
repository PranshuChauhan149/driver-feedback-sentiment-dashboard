🚗 Driver Feedback & Sentiment Dashboard

A modern React-based dashboard for collecting, analyzing, and visualizing driver feedback in real time.
This application helps organizations monitor driver performance, identify trends, and make data-driven decisions through structured feedback and sentiment analytics.

✨ Features

📊 Real-time performance dashboard

😊 Sentiment analysis (positive / neutral / negative)

🏆 Driver leaderboard with sorting & filtering

📝 Multi-step feedback submission form

🔍 Individual driver detail & trend analysis

⚡ Instant updates without page refresh

📱 Fully responsive design

🛠️ Tech Stack

React + TypeScript

Vite

Tailwind CSS

React Query

Zustand

Recharts

🎯 Use Case

Designed for corporate transport or fleet management systems to track driver performance, collect structured feedback, and improve service quality using analytics.
